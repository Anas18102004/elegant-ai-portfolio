

## Mohammad Anas — Premium AI/ML Portfolio

### Design System
- **Theme**: Dark mode base with blue/purple neon accents
- **Glass UI**: Frosted glass cards with blur, soft shadows, transparency
- **Typography**: Inter font, large bold headings, generous spacing
- **Colors**: Deep grey/black background, blue-purple gradient accents, subtle glows

### Pages & Layout

#### Hero Section
- Bold heading: "I build intelligent AI systems & scalable infrastructure"
- Subtitle: AI/ML Engineer | DevOps | Multi-Agent Systems
- Animated gradient mesh background with light floating particles
- Clean CTA buttons (View Projects / Contact)

#### Bento Grid Homepage
Apple-style grid blocks below the hero:
- **About Me** — short bio card with glassmorphism
- **Projects** — preview grid (3-4 featured)
- **Experience** — compact timeline preview
- **Skills** — interactive tag cloud
- **Contact** — minimal links block

#### Projects Section
- Grid of glassmorphism cards with hover scale + glow effects
- Each card shows project name, short description, tech tags
- Click opens a detailed modal/page: Problem → Solution → Tech → Impact

#### Experience Section
- Vertical timeline with glassmorphism cards
- WebXpress AI (DevOps, CI/CD, AWS, DigitalOcean)
- JBCODEAPP (AI/ML, healthcare system)

#### Skills Section
- Interactive glowing tags grouped by: AI/ML, Backend, DevOps, Tools
- Hover glow effect on each tag

#### Contact Section
- Clean minimal section with social links and email
- Simple contact form or direct links

### Animations (Framer Motion)
- Smooth fade-in on scroll for sections
- Card hover: subtle scale + border glow
- Button hover glow effect
- Light particle background (hero only)
- No heavy or distracting animations

### Technical
- Single-page app with smooth scroll sections
- Fully mobile responsive
- Fast loading, optimized animations
- Framer Motion for all transitions
